# User Guide

このガイドは、家庭内でアプリを使う人向けの説明です。

## 起動

開発中は次のURLで開きます。

```text
http://localhost:3000
```

Docker ComposeでLAN内サーバーとして起動した場合は、サーバー端末のIPアドレスを使います。

```text
http://<local-ip>:3000
```

## ログイン

初期状態ではseedでownerユーザーが作成されます。

```text
email: owner@example.local
password: change-me
```

実運用前に管理者は `.env.local` の値を変更し、DBを作り直してください。

## 画面

### 今日買う

現在の買い物リストです。Phase 1ではモックデータですが、チェック操作は画面上で動きます。

表示されるもの:

- 商品名
- 数量
- 店舗グループ
- 優先度
- メモ
- 完了済みの折りたたみ

### いつもの

定番品や消耗品の一覧です。Phase 1では静的表示です。

### 残量

残量が少ないもの、空に近いものを確認する画面です。Phase 1では静的表示です。

### 設定

現在のログインアカウント、家庭名、ロールを確認できます。ログアウトもここから行います。

## ログアウト

設定画面の「ログアウト」を押します。ログアウト後はログイン画面に戻ります。

## 現時点の注意点

Phase 1では、買い物リストや残量の変更はDBに保存されません。ブラウザを更新するとモックデータに戻ります。

実際の保存はPhase 2で実装します。
